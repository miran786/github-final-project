# Simple Interest Calculator

A simple interest calculator application.

## Introduction

This repository contains a Bash script that calculates **Simple Interest** based on user-provided inputs.

## What is Simple Interest?

Simple Interest is calculated using the formula:

**SI = (P × R × T) / 100**

Where:
- **P** = Principal amount
- **R** = Rate of interest (per annum)
- **T** = Time period (in years)

## Usage

Run the script and provide:
1. Principal amount
2. Rate of interest
3. Time period

The script will output the calculated simple interest.
